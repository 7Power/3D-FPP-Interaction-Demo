# 3D-FPP-Interaction-Demo
Godot 3.x First Person Perspective demo/template.

# Features:
* Movement (walking, sprinting, jumping, crouching)
* Using ladders
* Using stairs
* Interaction system
* Picking up (and throwing) objects
